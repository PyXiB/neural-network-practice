# README.md

## Описание проекта

Этот учебный проект направлен на разработку, обучение и тестирование нейронной сети для классификации изображений рукописных цифр из набора данных MNIST. Нейронная сеть создана с использованием библиотеки PyTorch. Основной задачей является изучение принципов построения и обучения нейронных сетей, включая работу с данными, подбор параметров обучения и оценку результатов.

## Цели проекта

- Освоение процесса разработки нейронной сети на Python с использованием PyTorch.
- Проведение экспериментов по выбору топологии и параметров обучения сети.
- Проверка качества работы модели на тестовых данных.
- Визуализация результатов предсказания и анализа потерь.

## Файловая структура

```
├── data/
│   └── MNIST/
│       └── raw/            # Исходные данные MNIST
├── main.py                  # Основной файл проекта
├── requirements.txt         # Список необходимых библиотек
```

## Используемые технологии

- **Python 3.8+** — язык программирования.
- **PyTorch** — библиотека для разработки и обучения нейронных сетей.
- **Torchvision** — библиотека для работы с популярными датасетами.
- **Matplotlib** — библиотека для визуализации.

## Особенности реализации

1. **Данные**: Используется набор данных MNIST, который содержит изображения рукописных цифр. Данные автоматически скачиваются и загружаются в папку `data/MNIST/raw`.
2. **Нейронная сеть**: Реализован многослойный перцептрон с двумя скрытыми слоями.
3. **Обучение**: Используется функция потерь CrossEntropyLoss и оптимизатор Adam. Параметры обучения (скорость, количество эпох) подобраны экспериментально.
4. **Визуализация**: Реализован вывод графика потерь на обучении и отображение примеров предсказаний модели.

## Установка

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/PyXiB/neural-network-practice.git
   ```
2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Использование

1. Запустите основной файл для обучения и тестирования модели:
   ```
   python main.py
   ```
2. В ходе выполнения программа выведет:
   - График изменения потерь на обучении.
   - Примеры предсказаний модели на тестовых данных.

## Результаты

- График потерь показывает снижение ошибки модели в процессе обучения.
- Примеры предсказаний визуализируют способность модели правильно классифицировать изображения цифр.
